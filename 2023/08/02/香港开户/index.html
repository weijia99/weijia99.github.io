香港银行开户

1.简介

在7.27号特种兵式旅游去香港来进行开户，主要开了以下几个账户

  1. 中银香港
  2. livi银行
  3. za银行
  4. 富途牛牛

失败的有：汇丰银行

2.事前准备

通用物品：

  1. 身份证
  2. 港澳通行证
  3. 过关小条
  4. 1k港币
  5. 住址证明（可选，如果使用的是身份证地址就不需要）

手机开通境外短信功能，避免接受不了香港短信（可以直接打10000来帮忙开通）

购买境外流量（我使用的是oppo的境外流量，12元10G，不过感觉500m之后就被限速了），可以直接开通境外漫游，这样就可以直接使用，一天25元封顶。

2.1中银香港

提前去中银香港公众号来进行预约开户，最少提前两天。

提前下号app：

  1. 中银香港（开户时需要进行填写资料，要在app上面填写）
  2. boc pay（可以当做云闪付在内地使用）

推荐网点：

  1. 太子分行
  2. 土瓜湾分行
  3. 网红中港城
  4. 奥海城二期
  5. 尖沙咀分行

可以多选几个作为备注。

我使用的是土瓜湾分行

2.2汇丰银行

汇丰我也是在汇丰香港微信公众号进行预约申请的，汇丰我预约了之后，过几天，会有客户经理打电话再次询问你开户情况，然后提醒你带资料。之后还有一个电话会来商量最后预定的时间。

3.开户教程

3.1旅途过程

我做的是从深圳北出发的高铁，直接到香港九龙高铁站。20分钟花了75，不得不说，香港真是寸土寸金。

之后就是进行香港过关，执行的是一地两检，首先刷身份证进行出站，之后就是如果是第一次来香港，就需要走人工通道，然后才能过关，走了人工之后，就可以以后走自主通道。我来的那一天是工作日周四，人还是比较少的。

过关之后，还需要使用港澳通行证出香港关口，这个时候会出现一个小票，要注意保留好，这个开户需要用到的。

3.2中银开户

过关之后，首先就是弄一个八达通（只接受现金，就是比较离谱的），八达通在香港的地位和微信支付宝差不多，吃饭地铁都可以用八达通（深圳口岸也可以给八达通进行充值，使用云闪付）。

之后就是坐地铁到土瓜湾，没有经过转车（主要是香港也不大），然后4站就好像是6块多（血鬼），然后按照导航走到土瓜湾中银。

到了中银，首先和大堂经理，表明，自己预约了的。她会记住你的预约成功的一个微信代码，然后让你下载app来进行扫码开户，由于我之前就下载好了，直接就进行扫描，然后正常的填写资料。之后让我等待。（顺带一提，大唐经理人挺好的，根本就没有为难，而且也没有歧视。她之前说的是粤语，后来跟我切换成普通话来进行交流）

那天开户网点的人也不多，好像就我一个，我来的时候，已经10点多了，我看网红中港城8点半就没有号了，所以还是选择一个偏僻一点的网点。

开户的柜员是个小哥，人也挺好的，也没问我开户原因，还问我开不开港股，跟我说了港股操作，只有股票账户有股票，就需要半年扣150的政府税，还有其他东西。反正20多分钟就开户完成了，很顺利，还主动问我我的邮寄地址和我的身份证地址是不是一致的，我说不是，他还主动帮我把地址改到我现在住的地方。

总的来说还是很顺利的，中银里面还有wifi，可以在等待的时候完成livi和za的开户。

物品总结：

  1. 身份证
  2. 过关小票
  3. 港澳通行证
  4. 学生证和在读证明我带了，没看
  5. boc hk
  6. boc pay
  7. 可以漫游的手机号

10点20到的，然后11点左右的时候就出发了。我以为都会想开头这么顺利，于是就去冲汇丰了。

失策的地方：没有在app上联系客服改dhl挂号信，现在是平信，我都不知道到哪里

3.3汇丰银行

本来来香港的时候就感觉开不了，不包多大的希望，但是中银开户的过于顺利，让我想盲一下。我预约的是12点来进行开户。本来预约的是土瓜湾分行，结果到了他哪里发现，汇丰把这个网点关了。。。。白跑了一趟，于是我按照指引去了红磡分行，感觉就是没法开的。

走到红磡分行的时候，正好也差不多到12点，有一说一，汇丰确实是香港第一大行，人比中银多了很多。到了之后，和经理说了开户，这个时候就感觉他有点歧视，也感觉不太顺利。给我取号之后，等了10几分钟就和汇丰经理来讨论。

看了我的身份证之后

  1. 问我开户目的是干什么
  2. 有没有投资经验
  3. 一个月来香港几次
  4. 一个月必须要来香港1-2次才可以，而且还要有投资经验

感觉就是故意刁难的，都还没问我是不是学生。于是就在10多分钟之后，就开户失败。可能是我去的银行人有点多吧。所以被拒了。

本来就是希望不太大的。群友也有被拒2次，然后才开户成功的。还是要去哪些容易去的网点，主要有以下几个

  1. 皇后大道
  2. 上水
  3. 尖沙咀（可选，群友在这里被拒）

经验可以参考有关这个视频：https://youtu.be/q_uNOF2z1R0

看下次有时间能不能去再开一次，毕竟汇丰开了就有一个MasterCard的提款卡

3.3za和livi

者两个在手机上就可以开，一个是虚拟visa，一个是虚拟master，za可以申请实体卡。

开户材料都是一样的：

  1. 连上香港网络
  2. 提交身份证
  3. 提交手机号
  4. 提交港澳通行证
  5. 打开手机定位，定位在香港
  6. 还需要提交出入境记录，这个za有教程

然后就是等了10多分钟就开户成功了。然后注册fps（这个稍后再讲）

3.4富途牛牛

由于众所周知的原因，牛牛现在在境内无法开户。但是到了香港，这一底线就灵活起来了。选择开户就进入到开户选项，这个时候需要进行选择，选择最后一个，内地用户存量开户。这就需要定位你人在香港，他说要提交香港港股账号的证明，正好我开的中银就提供了港股账号，第一次我提交的是我的开户纸，上面有我的开户信息还有股票信息，结果不通过，说要app截图，然后我就把我boc hk的股票账户截图发给他，之后过滤10多分钟就开户成功。

3.5开通fps

fps就是香港银行快速转账系统，不收取手续费的，这样我们就可以用boc hk来给livi和za如今。

开通教程很简单，直接搜索fps，然后进行登记手机号，就可以绑定银行了，之后转账都可以直接香港中银转到其他银行。

3.6境内转境外

这个时候就需要在境内有一个中银，因为中银转到境外的中银是面手续费的。

  1. 首先购汇-兑换港元
  2. 选择转账-跨境汇款
  3. 填写跨境资料-主义找一个酒店的地址，然后谷歌翻译成英文提交
  4. 一个月不要超过3次，不然会被中行非桂

3.7出境（重要）

出境，至少提前一个小时进入到西九龙高铁站，因为还要过海关安检，我因为带的东西比较多，还被海关叫住了，看了我的书包，才发现是药品还有吃的，然后说了依据不到5k，就让我走了。至少提前一个小时过安检，因为要检查两道。休息日可能人还是更多，序偶已得规划好时间，一天就是早上9点出发，然后下午4点回来。

4.事后

目前已经收到了za card，从珠海发货，还是很快的。中银是平信，不知道现在在哪里了。下面介绍几个可选选项

4.1中银加入apple pay

开户第二天进入boc hk选择设置，扫码提款，就可以看到提款卡的账号，然后输入到apple pay就可以加入了。

之后livi 还有za可以直接在app里面加入到apple pay

4.2激活wise

wise激活需要20usd进行如今验证，可以使用za livi来进行验证，我之前使用的是depay如今，损耗10个点才激活wise，不过现在wise好像也不好激活了。

4.3海淘

chatgpt订阅，还有oyunfor土耳其礼品卡，这些都是可以使用境外卡的，国内银联都是用不了的。

4.4中银境内付款

下载使用boc pay，然后绑定自己的中银账号，给当前boc pay进行充值，之后就可以使用boc pay扫描云闪付的二维码来进行付款，最后损耗差不多1个点的样子。

4.5WeChat香港钱包

首先联系微信记账本，然后输入香港两个字，之后找到开通香港钱包，然后输入za的银行卡账号，就可以绑定了。绑定之后选择钱包只有右上角的三个点可以切换回内地的钱包。每次消费2.5个点，有点多了

4.6zen充值

zen是一个立陶宛的虚拟卡，已经关门了，可以用绑定的apple pay给zen进行充值，za的转换费用还有苹果费用，大致扣了5-6个点。

5.总结

如果有想去香港玩的，还是建议办一张港卡，毕竟香港现在也在推广比特币，让银行接受比特币资金。玩港股也可以用港卡，还有netflix订阅，spotify订阅，谷歌play apple store，海涛等多方面，还是可以办一张的。
